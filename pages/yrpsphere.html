120-Day Curriculum

11 phases.
One complete engineer.

From "what is a computer?" to a full-stack banking portal with React + FastAPI + PostgreSQL.

Phase 1
Computers, Programming & Python Foundations
Day 1–18
  • How computers & operating systems work
  • Programming languages, compilers, interpreters
  • Python environment setup, first script
  • Variables, data types, operators, control flow
Phase 2
Python Basics — Strings, Collections, Functions
Day 9–18
  • String manipulation, f-strings, slicing
  • Lists, Tuples, Dicts, Sets
  • Functions, scope, higher-order functions
  • Modules & standard library
🏨 Project: Hotel Mgmt System
Phase 3
OOP & Intermediate Python
Day 19–27
  • Classes, objects, instance & class variables
  • Inheritance, encapsulation, polymorphism
  • Abstract methods, MRO, super()
  • Inner classes, is-a & has-a relationships
🏥 Project: Hospital Mgmt System
Phase 4
Advanced Python — Decorators, Generators, Files
Day 28–40
  • Decorators, closures, parameterized decorators
  • Generators, iterators, yield
  • Logging, exception handling, custom exceptions
  • File handling, Regex, memory management
🔐 RBAC System 🎟️ Dynamic Ticket Pricing
Phase 5
Major Python Project
Day 41–50
  • Full project lifecycle: requirements → planning → build
  • Modular OOP architecture, secure authentication
  • Regex validation, CLI enhancements
  • Persistent file storage, error handling
🏦 CLI Banking System
Phase 6
Concurrency, SQL & FastAPI Backend
Day 51–68
  • Multithreading, multiprocessing, async/await
  • SQL: DDL, DML, joins, aggregations
  • FastAPI: routing, Pydantic, dependency injection
  • PostgreSQL + SQLAlchemy ORM
👷 Employee/Task Manager API
Phase 7
HTML & CSS Fundamentals
Day 69–79
  • HTML structure, semantic elements, forms
  • CSS selectors, box model, flexbox
  • Responsive design with media queries
🏨 Hotel UI (Static)
Phase 8
JavaScript & DOM
Day 80–90
  • JS fundamentals, let/const, control flow
  • DOM manipulation, events, form validation
  • localStorage, fetch() API
🏨 Hotel App (Interactive)
Phase 9
React Core
Day 91–101
  • React + Vite setup, JSX, functional components
  • useState, props, event handling
  • Lists, forms, controlled components
🏥 Hospital App (React)
Phase 10
React + API + Advanced Patterns
Day 102–108
  • useEffect, data fetching, React Router
  • Context API, custom hooks (useAuth, useFetch)
  • Protected routes, React.memo, useCallback
Phase 11 · Capstone
Full-Stack Banking Portal
Day 109–120
  • FastAPI backend: JWT auth, RBAC, full CRUD
  • React frontend: dashboard, accounts, transactions
  • Transfer money, pagination, filters
  • Admin module, clean architecture
🏦 Bank of Python — Full-Stack Portal
